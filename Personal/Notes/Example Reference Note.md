# Example Reference Note

This is an example of how to structure a reference note in your vault. Use it as a template for your own notes.

## Summary

Every reference note starts with a brief summary — one or two sentences explaining what the note is about and why it exists.

## Details

The details section is where the main content lives. Write as much or as little as you need.

### Subsections

Use subsections (`###`) to break up longer notes into scannable chunks.

### Wikilinks

Link to related notes using double brackets: [[Getting Started]]. This creates a bidirectional link — both notes know about each other.

### Tags

Tags go after the main heading of a note (not in the frontmatter). Common examples:
- `#personal` — personal life
- `#learning` — learning and study
- `#project` — project-related
